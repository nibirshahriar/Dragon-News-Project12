# 📰The Dragon News

A modern and responsive news web application built with **Next.js 15**, **Tailwind CSS**, and **DaisyUI**.  
Users can browse news by category, view detailed articles, and enjoy a clean modern UI experience.

---

## ✨ Features

✅ Dynamic News Categories  
✅ News Details Page  
✅ Responsive Layout  
✅ Beautiful UI Design  
✅ Active Navigation Highlight  
✅ Breaking News Marquee  
✅ Loading Spinner  
✅ Sidebar Widgets  
✅ Dynamic Routing with Next.js  
✅ Clean Component Structure

---

## 🚀 Technologies Used

- ⚛️ Next.js 15
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🖼️ Next/Image
- 🔗 Next/Link
- 🎠 React Fast Marquee
- 🎯 React Icons
